---
title: 'Default Loan'
openapi: 'POST /default'
---

Marks a loan as defaulted.  
This endpoint is used when the borrower fails to make scheduled repayments beyond a grace period. Marking the loan as defaulted may trigger liquidation of collateral or other recovery processes depending on the agreement.

## Request Body

The request must include:

- `loanId`: The unique identifier of the loan being defaulted.

## Example Request

```bash
curl -X POST https://your-api.com/default \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "loanId": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
  }'
```

## Response

- Returns a success confirmation with the blockchain transaction hash.

```json
{
  "transactionHash": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
}
```
