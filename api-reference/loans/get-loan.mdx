---
title: 'Get Loan by ID'
openapi: 'GET /loans/{loan_id}'
---

This endpoint retrieves detailed information about a specific loan, including borrower profile and financial terms.  
It is ideal for powering dashboards, audit logs, or servicing platforms.

**What it does:**
- Fetches a loan record using its unique ID
- Includes financial metadata, repayment state, and borrower identity
- Displays servicing and underwriting progress for that loan

**When to use it:**
- When building a loan servicing dashboard
- When checking repayment status and borrower profile
- When you need underwriting or origination data
- When support teams need visibility into the loan lifecycle

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `loan_id` | string | Yes | The unique identifier of the loan to retrieve |

## Example Request

```bash
curl -X GET https://your-api.com/loans/0x1cb52e0315a448ddb84f1fb3016caf0686a864896dec79d7d89d1421d2ff208f \
  -H "Content-Type: application/json"
```
