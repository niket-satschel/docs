---
title: 'Request Collateral'
openapi: 'POST /request-collateral'
---

Initiates a request for collateral transfer from the borrower.  
This is typically triggered once underwriting has been completed and approved. The borrower is expected to transfer the required collateral assets as part of the loan process.

## Request Body

The request must include:

- `loanId`: The unique identifier of the loan for which collateral is being requested.

## Example Request

```bash
curl -X POST https://your-api.com/request-collateral \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "loanId": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
  }'
```

## Response

- Returns a success confirmation with the blockchain transaction hash.

```json
{
  "transactionHash": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
}
```
