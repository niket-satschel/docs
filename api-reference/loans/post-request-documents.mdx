---
title: 'Request Documents'
openapi: 'POST /request-documents'
---

This endpoint initiates a request for documentation associated with a specific loan. It is typically called by the loan servicer, lender, or platform to notify the borrower or relevant stakeholders to submit required documents as part of the loan origination process.

**What it does:**
- Triggers a formal request for loan-related documents
- Identifies required documentation based on loan type and underwriting needs
- Logs and tracks document request status in the system

**When to use it:**
- Immediately after loan creation and before verification
- To prompt borrowers or co-applicants to upload missing documentation
- As part of an automated or manual onboarding workflow

## Request Body

The request body should include the unique identifier of the loan for which documents are being requested:

```json
{
  "loanId": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
}
```

## Response

- Returns a confirmation of success along with the blockchain transaction hash:

```json
{
  "transactionHash": "0x5a0c0663a48e3abf16c7f0ee632b57da833e44e02a290fc0920ea2858da31f73"
}
```
